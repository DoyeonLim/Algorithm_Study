## 2주차 알고리즘 : Backtracking ( 백트래킹 )

- 해를 찾는 도중 해가 아니어서 막히면, 되돌아가서 다시 해를 찾아가는 기법
- 순차적 탐색을 가장 많이 사용하나, DFS ( 깊이 우선 탐색 ) 방식도 사용함
